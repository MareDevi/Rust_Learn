RCLI 项目介绍

RCLI 是一个使用 Rust 编写的命令行工具集合。本工具旨在提供高效、安全且易用的命令行体验。

功能特点

  • 快速执行常用命令
  • 支持自定义配置
  • 内置多种实用工具

使用方法

安装

cargo install rcli

基本命令

  • rcli help - 显示帮助信息
  • rcli version - 显示版本信息
  • rcli config - 配置设置

配置说明

配置文件位于 ~/.config/rcli/config.toml，您可以根据需要自定义各项设置。
